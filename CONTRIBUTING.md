# Contributing

Thanks for taking the time to contribute.

## Getting started

```bash
npm install
npm test
npm run typecheck
npm run lint
```

## Pull requests

- Keep changes focused — one topic per PR.
- Add or update tests for any behavior change.
- Make sure `npm test`, `npm run typecheck` and `npm run lint` pass.
- Clear commit messages (Conventional Commits style is appreciated).

## Reporting bugs

Open an issue with a minimal reproduction and the expected vs actual behavior.

Questions or custom work: [rovidev.com](https://rovidev.com).
